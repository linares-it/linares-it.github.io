# Spec Principal: Certificaciones

## Estado
**FREEZE — No implementar hasta tener credenciales verificables con badge Credly.**

## Visión (cuando se desbloquee)
Página `certificaciones.html` con una credencial por tarjeta, enlace directo al badge verificable en Credly.

## Condición de desbloqueo
Al menos una credencial verificable en alguna de estas plataformas:
- SAP Learning (badge Credly)
- Anthropic Skilljar
- Google Cloud Skills Boost
- AWS Partner Network
